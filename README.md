# ISBEROAL ERP - Agentes de Facturación

Agentes Python para procesamiento automático de facturas de ISBEROAL SL.

## Módulos

### Accounts Payable (Facturas de Gasto)
Agente que procesa facturas de gasto via OCR e imputa al ERP.
Ver [accounts-payable/README.md](accounts-payable/README.md)

### Accounts Receivable (Facturas de Cobro)
Agente que gestiona facturas emitidas y cuentas a cobrar.
Ver [accounts-receivable/README.md](accounts-receivable/README.md)

## Configuración
Copiar .env.example a .env y rellenar con las claves reales.
